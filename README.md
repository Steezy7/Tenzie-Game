# Tenzies Game 

A fun and interactive Tenzies dice game built with **Next.js** and **Tailwind CSS**. The game recreates the classic Tenzies experience with a modern web interface, allowing players to roll dice, hold selected values, and aim to match all dice in the fewest rolls and shortest time possible. This project demonstrates state management, game logic, and responsive UI design.

##  Features

- Roll dice and hold selected dice to strategize  
- Track the number of rolls and time taken  
- Reset game for unlimited play  
- Clean, responsive, and mobile-friendly design  

##  Tech Stack

- **Next.js** – React framework for server-side rendering and routing  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **JavaScript** – Core programming language for game logic
